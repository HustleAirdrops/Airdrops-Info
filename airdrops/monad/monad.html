Remix Deployment Details

  • Video Guide: View guide
    Follow the video tutorial for deploying contracts.
  • Deploy a Smart Contract Using Remix: Remix
    Use Remix to deploy your smart contract.
  • File Name: Gmonad.sol
    Name of the smart contract file.
  • Code: 
    // SPDX-License-Identifier: MIT

// Make sure the compiler version is below 0.8.24 since Cancun compiler is not supported just yet
pragma solidity >=0.8.0 <=0.8.24;

contract Gmonad { 
    string public greeting;

    constructor(string memory _greeting) {
        greeting = _greeting;
    }

    function setGreeting(string calldata _greeting) external {
        greeting = _greeting;
    }
}
    Smart contract code for deployment.
  • Deploy Name: Gmonad
    Name used for deploying the contract.
  • Greeting Message: gmonad Hello
    Initial greeting message for the contract.
